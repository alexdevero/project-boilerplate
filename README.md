### Install

Clone this repo
```
$ git clone https://github.com/alexdevero/project-boilerplate.git
```

### Usage
#### Run the project

```
npm run start
```
or
```
yarn start
```

#### Build the project

```
npm run build
```
or
```
yarn build
```

#### Test the project
```
npm run test
```
or
```
yarn test
```
